# AI-chatbot-for-Student-support-services
AI chatbot for educational institutions, offering automated responses to student inquiries and support services.
## 📌 Overview
AI Chatbot for Student Support Services is a machine learning-based chatbot that helps students by answering common queries related to college services such as admissions, fees, examinations, scholarships, library services, and general student support.

The chatbot uses Natural Language Processing (NLP) with TF-IDF Vectorization and the Multinomial Naive Bayes algorithm to classify user queries and generate relevant responses. A simple Gradio interface is used for interaction.

## ✨ Features

- 🤖 AI-powered student support chatbot
- 🎓 Answers admission-related queries
- 💰 Provides fee information
- 📝 Responds to examination questions
- 🎓 Scholarship guidance
- 📚 Library support
- 👨‍🎓 Student support services
- 🌐 User-friendly Gradio interface
- ⚡ Fast response using Machine Learning

## 🛠️ Technologies Used

- Python
- Pandas
- NLTK
- Scikit-learn
- Gradio

## 📂 Project Workflow

1. Install required libraries.
2. Create a dataset containing student questions and answers.
3. Preprocess the text using NLTK.
4. Convert text into numerical vectors using TF-IDF Vectorizer.
5. Train the chatbot using the Multinomial Naive Bayes algorithm.
6. Predict user intent based on the input query.
7. Display responses through a Gradio Chat Interface.
